# User guide screenshots

These assets are referenced from the guides as `./images/<filename>.png`.

To refresh them locally, run the stack in Docker (see root `README.md` and
`docs/operations/docker-user-guide.md`), open the app in a browser at your
configured `APP_URL` (default `http://localhost:3001`), and capture the same
views. Prefer full-page captures for dense dashboards when file size allows.
